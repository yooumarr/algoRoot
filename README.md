# AlgoRoot

The directory structure for the project looks like:

<pre>
algoroot/
├── functions.py # Predefined functions
├── rag.py       # RAG model
├── code_gen.py  # Code generation
├── api.py       # FastAPI API service
└── main.py      # Main file
</pre>
